# 小麦病虫害智能问答与学习平台 - 接口文档

---

## 1. 接口概述

### 1.1 基础信息

| 项目 | 说明 |
|-----|------|
| **API版本** | v1 |
| **基础路径** | `/api` |
| **协议** | HTTPS |
| **数据格式** | JSON |
| **字符编码** | UTF-8 |

### 1.2 状态码说明

| 状态码 | 说明 |
|-------|------|
| 200 | 请求成功 |
| 201 | 创建成功 |
| 400 | 请求参数错误 |
| 401 | 未认证或Token过期 |
| 403 | 无权限访问 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |

### 1.3 错误响应格式

```json
{
  "code": 400,
  "message": "错误描述信息",
  "data": null
}
```

### 1.4 认证方式

- **JWT Token**: 登录后获取Access Token，放在请求头 `Authorization: Bearer <token>`
- **游客访问**: 部分接口支持游客访问（有限制）

## 2. 认证模块

**POST** `/api/auth/register`

**请求体**:

| 字段 | 类型 | 必填 | 说明 |

|-----|------|-----|------|
